getHigherCard

fun getHigherCard(card1: Card, card2: Card, briscolaSuit: Suit): Card

Returns the higher card between two cards. If a card is the briscola, it is considered higher than any other card. If the two cards have different suits, the card with the first suit is returned. If the two cards have the same suit, the card with the highest value is returned. If the two cards have the same value, the card with the highest rank is returned.
